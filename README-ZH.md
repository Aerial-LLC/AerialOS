## 不支援的清單：
- 透過「setup.exe」/媒體建立工具（無用的Rufus/Ventoy或其他部署實用程式）安裝映像。
- Bitlocker 磁碟機加密
- 語音識別
- Windows 防禦者
- Windows 備份/還原點
- Windows 沙箱
- Windows 自動駕駛儀 <!-- 召回 -->
- 超V
- 通知
- NFC/POS 設備
- 科塔娜
- Windows 更新
#### 其他不需要公開宣傳的元件
